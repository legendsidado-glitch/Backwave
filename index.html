Backwave — Quick Help

  • Remember: Say "remember X is Y" or use "/remember key=value". Backwave saves it to memory.
  • Spend: Use "/spend 10 buy art" to instruct spending from wallet (simulated).
  • Chats: Create multiple chats to separate tasks and personas.
  • Voice: Toggle voice replies in the top bar, change voice in Settings.
  • External LLM: There's a hook `callExternalLLM()` in the code where you can plug in an API call.
